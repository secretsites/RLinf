示例库
===============

本节展示了 **RLinf 目前支持的示例集合**，  
展示该框架如何应用于不同场景，并演示其在实际中的高效性。  

目前，我们提供两大类示例：  

- **具身智能场景**：训练 **VLA** 模型用于具身智能。参见 :doc:`embodied`。  
- **推理场景**：训练 **LLM** 模型用于高级推理任务。参见 :doc:`reasoning`。  

示例库会随着时间不断扩展，涵盖新的场景和任务，  
以展示 RLinf 的多样性和可扩展性。
